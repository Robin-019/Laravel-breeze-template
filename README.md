# Laravel breeze template

## Installation

**STEP 1**
```bash
Create a new folder
```
**STEP 2**
```bash
git clone https://github.com/Robin-019/Laravel-breeze-template.git
```
**STEP 3**
```bash
cd <your folder name>
```
**STEP 4**
```bash
npm i
```
## How to run
**STEP 1**
```bash
npm run dev
```
**STEP 2: Open** *SECOND* **terminal in your vscode**
```bash
php artisan serve
```
**HAPPY CODING!**
